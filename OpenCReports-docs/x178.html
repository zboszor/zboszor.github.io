OpenCReport element

The toplevel <OpenCReport> element controls some global settings and serves as the topmost XML element for child elements.

Size unit attribute

size_unit="default"
size_unit="rlib"
size_unit="points"
The size_unit attribute specifies whether all size related settings in the report are in points, a.k.a. 1/72th inch.

The default is rlib which is the legacy RLIB behavior, leading to some inconsistency in size related settings. For example, width for field and literal (see later in the Output section) are in number of characters. This is influenced by the font size set for these text fields. On the other hand, width of horizontal lines and border width around some reports are specified in points.

RLIB compatibility attribute

rlib_compat="yes"
The rlib_compat attribute specifies whether RLIB compatibility is switched on.

It controls two subtle behavioural details in tabular report rendering with PDF output. See TODO.
